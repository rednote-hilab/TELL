# Memory

## Goal
Solve grid puzzle levels by arranging blocks so beam fills all Y-shapes with W during fire (space).

## Stable Rules
- **Controls**: Click to select block (turns Blue), arrows to move, space to fire, reset.
- **Fire**: Beams propagate from emitters, reflect off blocks, fill Y shapes. 35 frames when all Y filled.
- **Blocks cannot pass through each other or Y walls**.
